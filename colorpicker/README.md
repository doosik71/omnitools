# Color Picker

이 폴더에는 간단한 웹 기반 색상 선택 도구가 들어 있습니다.

## 사용법

`index.html`을 웹 서버(또는 로컬 파일)에서 열면 색상 팔레트를 보고 선택할 수 있습니다. 선택한 색상은 CSS나 HEX 코드로 복사할 수 있습니다.

## 개발

- 기본 HTML/CSS/JavaScript로 구현되었습니다.
- 필요 시 더 많은 색상 모드(HSLA, RGB 등)와 저장 기능을 추가할 수 있습니다.

## 기타

OmniTools 프로젝트의 다른 도구들과 함께 사용할 수 있도록 간단하고 상태(state-less)한 구조를 유지하는 것이 목표입니다.
