# OmniTools 웹사이트 도구 아이디어 목록

OmniTools는 **stateless**(서버 저장 없이 입력 즉시 처리) 도구 모음을 목표로 한다.  
아래는 우선 개발 후보 목록이다.

## 1) 텍스트/문서

1. 대소문자 변환기 (UPPER/lower/Title/Sentence)
2. 공백/줄바꿈 정리기 (trim, 중복 공백 제거)
3. 텍스트 정렬기 (오름/내림/중복 제거)
4. 단어/문자/줄 수 카운터
5. Markdown 미리보기 + HTML 변환
6. HTML ↔ Markdown 변환
7. 슬러그(Slug) 생성기
8. 랜덤 문장/단어 셔플러
9. 템플릿 치환기 (`{{name}}` 방식)
10. 정규식(Regex) 테스트러

## 2) 개발자 인코딩/포맷

1. JSON 포매터/압축/검증기
2. YAML ↔ JSON 변환기
3. XML 포매터/검증기
4. Base64 인코드/디코드
5. URL 인코드/디코드
6. JWT 디코더 (서명 검증 제외)
7. UUID 생성기 (v4)
8. 해시 생성기 (MD5/SHA-1/SHA-256)
9. HMAC 생성기
10. URL QueryString 파서/빌더

## 3) 웹/프론트엔드

1. CSS Box Shadow 생성기
2. CSS Gradient 생성기
3. CSS Clamp() 폰트 계산기
4. 반응형 브레이크포인트 미리보기
5. Open Graph/Twitter 메타 태그 생성기
6. Favicons 태그 생성 보조기
7. Color Contrast 검사기 (WCAG)
8. Tailwind 클래스 플레이그라운드
9. SVG 최적화/정리기
10. SVG Path 뷰어

## 4) 데이터/변환

1. CSV ↔ JSON 변환기
2. TSV ↔ CSV 변환기
3. 표(Table) → Markdown Table 변환
4. UNIX Timestamp ↔ 날짜 변환
5. 시간대 변환기 (Timezone converter)
6. 단위 변환기 (길이/무게/온도)
7. 진법 변환기 (2/8/10/16)
8. 숫자 포맷터 (천단위, 통화형)
9. 랜덤 데이터 생성기 (이름/이메일/ID)
10. Lorem Ipsum 생성기

## 5) 보안/네트워크

1. 비밀번호 강도 검사기
2. 강력한 비밀번호 생성기
3. SPF/DKIM/DMARC 레코드 설명기 (입력 파싱)
4. User-Agent 파서
5. HTTP 헤더 포맷/설명기
6. CIDR 서브넷 계산기
7. IPv4/IPv6 변환·표현 도구
8. MAC 주소 포맷 변환기
9. CORS 시나리오 설명기
10. CSP 정책 템플릿 생성기

## 6) 이미지/미디어 (브라우저 내 처리)

1. 이미지 리사이저
2. 이미지 포맷 변환기 (PNG/JPEG/WebP)
3. 이미지 압축기
4. EXIF 메타데이터 뷰어
5. 색상 팔레트 추출기
6. 파비콘 생성기 (기본 세트)
7. QR 코드 생성기
8. 바코드 생성기 (Code128/EAN)
9. 이미지 크롭 도구
10. 이미지 Base64 변환기

## 7) 생산성/기타

1. 체크리스트 생성기 (Markdown 출력)
2. 회의록 템플릿 생성기
3. 랜덤 추첨기 (명단 기반)
4. 간단 타이머/스톱워치
5. 나이/디데이 계산기
6. VAT/세율 계산기
7. 할부/이자 계산기
8. BMI/기초 건강 계산기
9. 의사결정 룰렛
10. 키보드 단축키 치트시트 생성기

## 우선순위 제안 (MVP 10개)

1. JSON 포매터/검증기
2. Base64 인코드/디코드
3. URL 인코드/디코드
4. UUID 생성기
5. 해시 생성기
6. CSV ↔ JSON 변환기
7. UNIX Timestamp ↔ 날짜 변환
8. QR 코드 생성기
9. 비밀번호 생성기
10. Color Contrast 검사기
